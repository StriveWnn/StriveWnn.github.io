Xshell命令

Created2022-09-27|Updated2022-10-17
|Post View:

man:查看某个命令的帮助（如:man ls）
touch:创建文本（如：touch test01.txt）
rm:删除文本（如：rm test01.txt）
mkdir:创建文件夹（目录,如:mkdir Test）
rm -rf:删除目录（文件夹,如：rm -rf Test）
rm *:删除当前目录下所有文件
rmdir:删除空目录（不是空目录删除不掉,如：rmdir Test）
find:查找文件或目录（如：find Test/test01.txt）
zip test01.zip /etc/services ：压缩文件
unzip test01.zip：解压文件
mv 原文件名 重命的名称：重命名文本或文件夹（目录,如：mv Test Tests）
mv 文件 需要移动到的目录：移动文件或文件夹（目录,如：mv Test F:\sdf\fafa）
vi:编辑文本（如：vi test.txt）
i：进入编辑模式
esc：退出编辑模式（键盘左上角esc按键 ）
:wq：退出并保存
:q!：退出不保存
more:查看文本内容（如：vi test.txt）
less:查看日志
ps -ef:查看进程
pwd:查看当前路径
sudo rz:windows上传文件到服务器
sudo sz 文件名:服务器下发到windows
top:查看cpu,内存
alt+回车：全屏
alt+1…:切换窗口
clear：清屏
free -tm：查看内存和swap分区使用情况
iostat:磁盘吞吐量
-c:只显示CPU行
-d:显示磁盘行
-k:以千字节为单位显示磁盘输出
-t:在输出中包括时间戳
-x:在输出中包括扩展的磁盘指标
last:查看最近用户登录情况
reboot:重新启动计算机
shutdown:停止服务，重启计算机
shutdown -h now：停止服务，关闭系统
halt：关闭计算机
su - :切换到root用户（需要输入当前账号的密码）
su – zhoulj:切换到zhoulj用户（切换到普通用户zhoulj）
注意：-，他很关键，使用-，将使用用户的环境变量

部署项目常用：
Ps –ef|grep xxxx:查看要xxxx进程的进程号（pid）
kill -9 pid:杀死pid的线程（pid:进程号）
lsof -i:端口 //查端口进程
nohup java -jar xx.jar &：启动jar包（方式一，关闭xshell不会导致程序挂掉，程序会在后台跑。日志输出到日志文件。）
java -jar xx.jar >nohup.out &：启动jar包（方式二，关闭xshell会导致程序挂掉，程序不会在后台跑。可以打印日志。）
tail -f 日志文件名：打印log日志（tail -f nohup.out）

操作虚拟机防火墙命令：
关闭虚拟机防火墙：systemctl stop firewalld（windows浏览器访问虚拟机web项目需要关闭虚拟机防火墙）
查看防火墙服务状态：systemctl status firewalld
开启防火墙：service firewalld start
关闭防火墙：service firewalld stop
重启防火墙：service firewalld restart
永久关闭防火墙：systemctl disable firewalld.service（慎用）

cmd命令：
mstsc ：启动远程桌面
ipconfig : ip查询
wscui.cpl ：Windows安全中心
desk.cpl ：显示属性
cleanmgr ：磁盘清理工具
devmgmt.msc ：设备管理器
compmgmt.msc ：计算机管理
dcomcnfg ：组件服务
control admintools ：管理工具
appwiz.cpl ：添加或删除程序
firewall.cpl ：Windows防火墙
write ：写字板
calc : 计算器
sysdm.cpl ：系统属性
